Before you begin

This is an F# language class. To complete the coding exercises, you will need a Xamarin development environment with either a simulator/emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This session shows you how to create, organize and build the basic components of an F# project.

Objectives

  1. Breakdown the structure of an F# program
  2. Identify the components of an F# program

Lecture

Projects in F#

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Reorder an F# project
    Exercise
    In this first lab, you will reorder the files in an iOS or Android project until it compiles successfully
  2. Use a NuGet component to combine images into a PDF
    Exercise
    In this exercise, you will port a C# program to F# using a NuGet component to generate a PDF file from a set of images.
  3. Create and run a console app
    Exercise
    In this last exercise, you will create modules as well as how to reference and organize code files in F# to create a console application.
